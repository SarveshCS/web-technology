Layer 6: Presentation Layer

The translator of the network

What is the Presentation Layer?

The Presentation Layer is responsible for data translation, encryption, and compression. It acts as a translator between the application layer and the network.

Think of it as the layer that ensures data is in a format that both the sender and receiver can understand.

Key Functions

  • Data Translation: Converts data between different formats (ASCII, EBCDIC, Unicode)
  • Encryption/Decryption: Secures data for transmission (SSL/TLS)
  • Compression: Reduces data size for faster transmission
  • Serialization: Converts objects to byte streams

Common Protocols & Standards

  • SSL/TLS - Secure Socket Layer / Transport Layer Security
  • JPEG, GIF, PNG - Image formats
  • MPEG, MP4 - Video formats
  • ASCII, Unicode - Character encoding
  • XML, JSON - Data serialization formats

Real-World Example

When you visit a secure website (HTTPS):

1. Your browser negotiates an SSL/TLS connection

2. Data is encrypted before transmission

3. Images are decompressed from JPEG format

4. Text is decoded from UTF-8 encoding
