OSI Model - All 7 Layers

Complete overview of each layer and its responsibilities

Layer Comparison Table

Layer Name Function Protocols/Examples Data Unit
L7 Application User interface & application services HTTP, FTP, SMTP, DNS Data

How Data Flows

Sending Data (Encapsulation)

When you send data, it goes DOWN the layers:

  • Layer 7: Application creates the data (e.g., web page request)
  • Layer 6: Data is formatted/encrypted
  • Layer 5: Session is established
  • Layer 4: Data is segmented, port numbers added
  • Layer 3: IP addresses added (packet created)
  • Layer 2: MAC addresses added (frame created)
  • Layer 1: Converted to electrical/light signals

Receiving Data (De-encapsulation)

When receiving data, it goes UP the layers:

  • Layer 1: Signals received and converted to bits
  • Layer 2: Frame checked, MAC verified
  • Layer 3: Packet routed by IP address
  • Layer 4: Segments reassembled
  • Layer 5: Session maintained
  • Layer 6: Data decrypted/decompressed
  • Layer 7: Data delivered to application
